# SAOIcons
SAO Utils Icons SAO 图标合集

# 概述 Summary
本项目旨在集成并分享SAO Utils中的图标，省去自制图标的痛苦。<br>
This repository is for collecting and sharing the icons within SAO Utils, so that you do not need to make them by yourself.
